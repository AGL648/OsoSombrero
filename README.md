# Oso & Sombrero
Example of a Text Based Type of Game

# Description
This is a Text Based Type of Game with the intention to be played locally and in the browser. The player must make a lot of different choices during the game with the aim to help the character Oso to get his hat back, and depending on the choices you make you'll get different endings.

It uses the numerical keys and others indicated during the levels in order to advance in the adventure.

# Features
- You can play it in your browser
- Multiple endings depending on the choices you make in the adventure.

# Programs Used
- Unity 6
- C# (Visual Studio)

# Lessons
I've learned how to make a text based web browser game with different choices and endings.
